1) Clone git repo here:

https://github.com/libjpeg-turbo/libjpeg-turbo/

2) Install CMake from here:

https://cmake.org/download/

3) Run
 cmake -G"Visual Studio 16" -A"Win32"

This will produce a .sln and several vcproj files to build using Visual Studio IDE.

4) Build the the libjpeg-static library, this is the only one that is needed to link with Studio.
